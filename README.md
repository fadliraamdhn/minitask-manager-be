
# Minitask Manager API

An API for providing system for minitask manager 

## Installation

Install Minitask Manager with npm

```bash
    git clone <repository-url>
    cd mini_task_manager
    npm install
```

Make a .env file in root project and add this code to run this project

```bash
DATABASE_URL=your-database-connection
JWT_SECRET=your-jwt-secret
CORS=your-frontend-url
```

Now run locally this project to development in your project

```bash
npm run start
```
